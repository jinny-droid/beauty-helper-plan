채널별 상품 분포 카드

7개 채널별로 아모레 상품 수를 가로 막대로 비교. 채널 이름 옆 막대 길이로 상대적 규모 직관적 파악.

정렬 기준
  • 고정 순서 — 네이버 → 쿠팡 → 올리브영 → 지마켓 → 11번가 → 카카오 → 옥션. 사용자 정렬 변경 불가.

우측 상단 "채널별 →" 링크 → 성과 분석 > 채널별 탭 이동

아모레 브랜드 현황 카드

16개 AP 브랜드 중 상위 6개 브랜드의 상품 수를 막대로 표시. 1~3위 브랜드는 시각적으로 강조.

정렬 기준
  • 전 채널 합산 상품 수 기준 내림차순. 상위 6개만 표시. 사용자 정렬 변경 불가.

"+ 10개 브랜드 더보기" 링크 → 성과 분석 > 브랜드별 탭 이동

경쟁사 주요 동향 카드

경쟁사 탭 동향 감지 결과 중 "주시 필요" 등급 경쟁 브랜드만 최대 4개 요약 노출.

노출 순서
  • 경쟁사 탭 테이블의 원본 순서 그대로 유지 (별도 정렬 없음)
  • 주시 필요 브랜드가 4개 미만이면 해당 수만 노출
표시 내용설명
브랜드명경쟁사 이름
동향 텍스트예: "쿠팡 신규 입점", "점유율 급등 · 주시 필요"
점유율 전월 대비증감 수치
기획 의도: 경쟁사 탭을 열지 않아도 위협적인 경쟁사 움직임이 있으면 Overview에서 바로 눈에 띄게. "경쟁사 →" 링크로 경쟁사 탭 전체 이동.

크로스채널 브랜드 검색

브랜드명을 입력하면 해당 브랜드 상품이 어느 채널에서 얼마에 판매되는지 채널별로 묶어 보여준다.

검색 조건
  • 검색어 입력 후 "비교" 버튼 클릭 또는 Enter 키로 실행 (실시간 자동 검색 아님)
  • 공백만 입력된 경우 검색 실행 안 됨
  • 검색 대상: 상품명 기준 부분 일치, 대소문자 구분 없음
  • 결과: 채널별 그룹으로 표시 (채널 고정 순서: 네이버 → 쿠팡 → … → 옥션)
  • 결과 상품 클릭 → 상품 상세 팝업
  • 결과 없을 경우 "해당 브랜드를 찾을 수 없습니다" 안내
