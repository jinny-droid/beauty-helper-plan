# 성과 분석 > 카테고리별 — 화면 기획

---

## 화면 목적

화장품 카테고리(스킨케어, 메이크업 등) 단위로 브랜드 간 경쟁 구도를 파악한다.  
"이 카테고리에서 우리 브랜드와 경쟁사가 어떻게 경쟁하고 있나"를 리뷰 점유율 기준으로 비교하는 화면.

---

## 화면 구성

```
┌─────────────────────────────────────────┐
│  서브탭: [채널별] [브랜드별] [카테고리별]  │
├─────────────────────────────────────────┤
│  카테고리 선택 탭 × 8                    │
├─────────────────────────────────────────┤
│  [선택 카테고리] 카테고리 브랜드 순위표    │
│  (AP + 경쟁사 포함)                      │
└─────────────────────────────────────────┘
```

---

## 카테고리 선택

8개 카테고리 탭 중 하나를 선택하면 해당 카테고리의 브랜드 성과표가 바뀐다.

**카테고리 목록:**  
스킨케어 / 메이크업 / 헤어케어 / 바디케어 / 선케어 / 향수 / 네일 / 맨즈케어

- 진입 시 기본 선택: 스킨케어
- 활성 탭: 네이비 배경 + 흰 글자 / 비활성 탭: 흰 배경 + 테두리

---

## 카테고리 브랜드 순위표

선택한 카테고리에서 AP 브랜드와 경쟁 브랜드를 함께 나열해 경쟁 구도를 한눈에 파악.

| 항목 | 설명 |
|------|------|
| 상품 수 | 해당 카테고리 내 등록 상품 수 |
| 리뷰 점유율 | 카테고리 전체 브랜드 리뷰 합산 대비 비율 |
| 총 리뷰 | 해당 카테고리 누적 리뷰 수 |
| 평균 판매가 | 해당 카테고리 할인가 평균 |

### 정렬 기준

**기본 정렬:** 리뷰 점유율 기준 내림차순 고정. 카테고리를 바꿔도 동일 기준 적용.  
사용자가 정렬 기준을 변경하는 옵션 없음.

### 스타일 규칙

| 대상 | 표시 방식 |
|------|----------|
| AP 브랜드 행 | 파란 배경 강조 + AP 배지 표시 + 네이비 텍스트 |
| 경쟁사 행 | 흰 배경 + 회색 텍스트 |
| AP 리뷰 점유율 | 네이비 배경 배지 |
| 경쟁사 리뷰 점유율 | 회색 배경 배지 |

AP 브랜드와 경쟁사를 시각적으로 즉시 구분할 수 있도록 행 색상으로 차별화.

| 네일 | 에뛰드 |
| 맨즈케어 | 아이오페 |

## 화면 흐름

```
카테고리별 탭 진입
  └── 카테고리 탭 클릭 → 해당 카테고리 브랜드 순위표 갱신
        └── 브랜드별 리뷰 점유율 · 상품 수 · 평균가 비교
```

## 미결 / 추후 개선

- 현재 카테고리 데이터 고정 → 실서비스 시 API 연동으로 동적 갱신
- 카테고리별 상위 상품 클릭 → 상품 상세 팝업 연결 현재 미구현 (텍스트만 표시)
