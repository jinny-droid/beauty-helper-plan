# Overview 탭 — 화면 기획

---

## 화면 목적

처음 대시보드에 진입했을 때 "지금 시장이 어떻게 돌아가고 있는가"를 30초 안에 파악할 수 있도록 핵심 수치와 동향을 한 화면에 압축해 보여준다.  
세부 분석 탭으로 이동하기 전 전체 맥락을 잡는 입구 역할.

---

## 화면 구성

```
┌─────────────────────────────────┐
│  핵심 지표 카드 × 4              │
├──────────────┬──────────────────┤
│ 채널별 상품  │ 아모레 브랜드    │
│ 분포         │ 현황             │
├──────────────┼──────────────────┤
│ 리뷰 급증    │ 경쟁사 주요 동향 │
│ TOP 5        │                  │
└──────────────┴──────────────────┘
```

---

## 핵심 지표 카드 (상단 4개)

화면 최상단에 시장 전반과 아모레퍼시픽 현황을 숫자로 빠르게 보여주는 요약 카드.

| 카드 | 표시 내용 | 클릭 시 |
|------|-----------|---------|
| 화장품 전체 상품 수 | 7개 채널에 등록된 화장품 상품 총합 | 이동 없음 (참고용) |
| 아모레 상품 수 | AP 브랜드 상품만 집계 | 성과 분석 > 브랜드별로 이동 |
| 총 리뷰 수 | AP 상품 전체 누적 리뷰 | 성과 분석 > 채널별로 이동 |
| 아모레 채널 점유율 | AP의 채널 내 리뷰 점유율 + 전월 대비 증감 | 성과 분석 > 채널별로 이동 |

### 기획 의도

- '전체 상품 수'는 시장 규모 파악용으로만 제공, 클릭해도 이동 없음
- 나머지 3개 카드는 클릭하면 해당 세부 분석 화면으로 바로 이동 → 관심 지표에서 즉시 드릴다운 가능
- 점유율 카드에 전월 대비 화살표 포함 → 추세를 숫자 하나로 전달

---

## 채널별 상품 분포 카드

7개 채널별로 아모레 상품이 몇 개 등록되어 있는지 가로 막대로 비교.

- 채널 이름 옆 막대 길이로 상대적 규모 직관적으로 파악
- 우측 상단 "채널별 →" 링크로 성과 분석 > 채널별 탭으로 이동

### 사용자 시나리오

> "어느 채널에 우리 상품이 가장 많이 올라가 있지?" 를 한눈에 확인하고 싶을 때

---

## 아모레 브랜드 현황 카드

16개 AP 브랜드 중 상위 6개 브랜드의 상품 수를 막대로 표시.  
1~3위 브랜드는 시각적으로 강조.

**정렬 기준:** 전 채널 합산 상품 수 기준 내림차순. 상위 6개만 표시.

- 하단 "+ 10개 브랜드 더보기" 링크 → 성과 분석 > 브랜드별 탭으로 이동

### 사용자 시나리오

> "어느 브랜드가 가장 상품 수 많은지" 대략 파악하고 브랜드 분석으로 바로 넘어갈 때

---

## 리뷰 급증 TOP 5 카드

이번 주 전 채널 합산 기준으로 리뷰가 가장 많이 증가한 상품 5개를 순위로 나열.

| 표시 내용 | 설명 |
|-----------|------|
| 순위 | 1~3위 로즈 색 강조 |
| 상품명 | 클릭 시 상품 상세 팝업 열림 |
| 카테고리 / 브랜드 | 맥락 정보 |
| 이번 주 증가량 | "+OOO" 형태로 얼마나 늘었는지 |

**정렬 기준:** 리뷰 증가량(delta) 기준 내림차순. 고정값으로 노출되며 사용자가 정렬 변경 불가.

- "트렌딩 →" 링크로 트렌딩 탭 전체 목록 이동 가능

### 사용자 시나리오

> 월요일 아침, 지난 주에 갑자기 주목받은 상품이 뭔지 바로 확인하고 싶을 때  
> 해당 상품 클릭 → 상세 팝업에서 리뷰 급증 시점과 가격 변동 연계해 파악

## 경쟁사 주요 동향 카드

경쟁사 탭의 동향 감지 결과 중 "주시 필요" 등급을 받은 경쟁 브랜드만 최대 4개 골라 요약 노출.

**노출 순서:** 경쟁사 탭 테이블의 원본 순서 그대로 유지 (별도 정렬 없음). 주시 필요 브랜드가 4개 미만이면 해당 수만 노출.

| 표시 내용 | 설명 |
|-----------|------|
| 브랜드명 | 경쟁사 이름 |
| 동향 텍스트 | 예: "쿠팡 신규 입점", "점유율 급등 · 주시 필요" |
| 점유율 전월 대비 | 증감 수치 |

- "경쟁사 →" 링크로 경쟁사 탭 전체로 이동

### 기획 의도

경쟁사 탭을 굳이 열지 않아도 위협적인 경쟁사 움직임이 있으면 Overview에서 바로 눈에 띄게.  
주시 필요 브랜드가 없으면 해당 카드는 빈 상태로 표시.

## 화면 흐름

```
Overview 진입
  ├── 핵심 지표 카드 클릭 → 성과 분석 탭 해당 서브탭으로 이동
  ├── 채널별 / 브랜드 카드 링크 → 성과 분석 탭 이동
  ├── 리뷰 급증 상품 클릭 → 상품 상세 팝업
  ├── 트렌딩 → → 트렌딩 탭
  └── 경쟁사 → → 경쟁사 탭
```
