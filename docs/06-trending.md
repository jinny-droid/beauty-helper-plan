# 트렌딩 탭 — 화면 기획

---

## 화면 목적

이번 주 리뷰 또는 찜이 급증한 상품과 주목받는 성분을 파악한다.  
"요즘 소비자들이 어떤 상품·성분에 관심을 갖고 있나"를 모니터링해, 상품 기획이나 마케팅 방향 설정에 활용하는 화면.

---

## 화면 구성

```
┌─────────────────────────────────────────┐
│  필터: 채널 | 지표(리뷰/찜) | 카테고리 | 아모레만 │
│  기간 표시: 2026-03-25 → 2026-04-01     │
├─────────────────────────────────────────┤
│  이번 주 주목 성분 카드 × 6 (3열)        │
│  또는 "성분 분석 미제공" 안내             │
├─────────────────────────────────────────┤
│  리뷰/찜 급증 상품 순위표                 │
│  [더 보기 버튼]                           │
└─────────────────────────────────────────┘
```

---

## 필터

| 필터 | 기능 |
|------|------|
| 채널 | 전체 또는 개별 채널 선택 (현재 프로토타입에서는 채널별 데이터 미분리) |
| 지표 | "리뷰 급증" 또는 "찜 급증" 전환 → 성분 섹션 제목과 테이블 헤더가 함께 바뀜 |
| 카테고리 | 전체 또는 8개 카테고리 중 선택 → 성분 섹션 노출 여부에 영향 |
| 아모레만 보기 | ON 시 순위표에 AP 브랜드 상품만 표시 |

필터 바 하단에 현재 조회 기간 표시 (예: "2026-03-25 → 2026-04-01 기간 대비 증가량")

---

## 이번 주 주목 성분 섹션

### 노출 조건

카테고리 필터가 성분 분석을 지원하는 카테고리인 경우에만 노출됨.

| 카테고리 | 성분 분석 |
|----------|----------|
| 스킨케어, 바디케어, 선케어, 헤어케어, 맨즈케어 | 제공 |
| 메이크업, 향수, 네일 | 미제공 → "선택한 카테고리는 성분 분석을 제공하지 않습니다" 안내 카드 표시 |
| 전체 | 제공 |

### 성분 카드 구성 (6개, 3열 그리드)

이번 주 리뷰 증가량이 높은 상위 6개 성분을 카드로 표시.

| 표시 내용 | 설명 |
|-----------|------|
| 순위 | 로즈 색 강조 |
| 성분명 | 예: 글루타치온, 나이아신아마이드 |
| 증가율 배지 | 30% 이상: 빨강 / 15~30%: 주황 / 15% 미만: 노랑 |
| 관련 상품 수 | 이 성분을 포함한 상품 수 |
| 브랜드 수 | 이 성분을 취급하는 브랜드 수 |
| 대표 상품명 | 이 성분 관련 상품 중 리뷰 증가량 1위 상품 |
| AP 포함 여부 | AP 브랜드가 이 성분을 취급 중이면 배지 표시 |
| 카테고리 | 이 성분의 주요 카테고리 |

### 사용자 시나리오

> "요즘 소비자들이 어떤 성분에 관심 갖는지 트렌드를 파악하고 싶다"  
> "글루타치온이 요즘 핫하다면 우리 브랜드 중 이 성분 포함 상품이 있는지 확인하고 싶다"  
> 상품 기획 회의 전 시장 트렌드 성분 파악용으로 활용

---

## 리뷰 / 찜 급증 상품 순위표

이번 주 리뷰(또는 찜) 증가량 기준 상위 상품을 순위로 나열.  
지표 필터(리뷰/찜)에 따라 표시 데이터와 컬럼명이 전환됨.

**정렬 기준:** 증가량(delta) 기준 내림차순 고정. 사용자가 정렬 변경 불가.  
**아모레만 보기 필터:** AP 브랜드 상품만 표시. 필터링 후에도 정렬 기준은 동일(증가량 내림차순), 순위 번호는 필터된 목록 기준으로 1번부터 재시작.

| 항목 | 설명 |
|------|------|
| 순위 | AP 상품: 로즈 / 경쟁사 상품: 네이비 |
| 상품명 | 길면 말줄임 표시. AP 배지 포함 |
| 브랜드 | 브랜드명 |
| 카테고리 | 상품 카테고리 |
| 현재 리뷰(찜) | 이번 주 기준 누적 수 |
| 이전 리뷰(찜) | 지난 주 기준 누적 수 (회색) |
| 증가량 | 이번 주 증가분 (+OOO) |
| 증가율 | 증가율 배지. 4% 초과: 빨강 / 이하: 주황 |
| 판매가 | 현재 판매가 |

- 행 클릭 시 상품 상세 팝업 열림
- AP 상품 행: 파란 배경 / 경쟁사 행: 흰 배경으로 시각 구분
- "더 보기" 버튼: 현재 프로토타입에서는 토스트 메시지만 표시 (추후 구현 예정)

### 사용자 시나리오

> "이번 주 리뷰가 가장 많이 늘어난 상품이 뭔지 확인하고 싶다"  
> "찜 급증 상품 중 경쟁사 제품이 뭔지 보고 싶다"  
> "아모레만 보기 ON → AP 브랜드 중 이번 주 주목받은 상품만 골라보기"  
> 해당 상품 클릭 → 팝업에서 리뷰 증가 시점과 가격 변동 연계 확인

---

## 화면 흐름

```
트렌딩 탭 진입
  ├── 지표 전환 (리뷰/찜) → 성분 섹션 제목 + 테이블 갱신
  ├── 카테고리 필터 → 성분 섹션 노출/미제공 안내 전환
  ├── 아모레만 보기 → 순위표 필터링
  └── 순위표 상품 행 클릭 → 상품 상세 팝업
```

---

## 미결 / 추후 개선

- 채널 전환 시 데이터가 분리되지 않음 (모든 채널에서 동일 데이터 사용)
- "더 보기" 버튼 실제 추가 로드 미구현 → 현재 안내 메시지만 표시
- 기간 텍스트 고정 → 실서비스 시 전주 기준 자동 계산으로 대체
- 성분 카드 6개 고정 → 실서비스 시 API 연동해 동적으로 갱신
