# 성과 분석 > 채널별 — 화면 기획

---

## 화면 목적

7개 이커머스 채널에서 아모레퍼시픽 상품이 어떤 성과를 내고 있는지 채널 단위로 비교하고 분석한다.  
"어느 채널에서 잘 되고 있나", "각 채널에서 우리 브랜드 순위는 어떻게 되나"를 파악하는 것이 핵심.

---

## 화면 구성

```
┌─────────────────────────────────────────┐
│  서브탭: [채널별] [브랜드별] [카테고리별]  │
├─────────────────────────────────────────┤
│  채널 선택 탭 (전체 + 7개 채널)  [Excel]  │
├─────────────────────────────────────────┤
│  ↓ 전체 선택 시                          │
│  전체 채널 비교 표                        │
│  인사이트 요약 카드 2개                   │
│  크로스채널 브랜드 검색                   │
├─────────────────────────────────────────┤
│  ↓ 개별 채널 선택 시                     │
│  해당 채널 KPI 카드 × 4                  │
│  아모레 브랜드 순위표                     │
└─────────────────────────────────────────┘
```

---

## 채널 선택

화면 상단의 채널 탭을 클릭해 조회 범위를 결정한다.

- **전체:** 7개 채널을 한 표에서 나란히 비교
- **개별 채널 (네이버 / 쿠팡 / 올리브영 / 지마켓 / 11번가 / 카카오 / 옥션):** 해당 채널만 집중 분석

Excel 다운로드 버튼으로 현재 화면 데이터를 엑셀 파일로 저장 가능.

---

## 전체 채널 비교 (전체 탭 선택 시)

### 전체 채널 비교 표

7개 채널을 한 행씩 나열하고 채널별 성과를 나란히 비교.

| 항목 | 설명 |
|------|------|
| 아모레 상품 수 | 해당 채널에 등록된 AP 상품 수 |
| 리뷰 점유율 | 채널 전체 화장품 리뷰 중 AP 상품이 차지하는 비율. 1위 채널은 강조 |
| 총 리뷰 | AP 상품 전체 누적 리뷰 수 |
| 평균 판매가 | AP 상품 평균 할인 판매가 |
| 평균 할인율 | 정가 대비 평균 할인 비율 |

**정렬 기준:** 고정 순서 — 네이버 → 쿠팡 → 올리브영 → 지마켓 → 11번가 → 카카오 → 옥션. 사용자가 정렬 변경 불가.

각 채널 행 우측 "상세 →" 를 클릭하면 해당 채널 단독 분석 화면으로 이동.

### 인사이트 요약 카드

비교 표 아래, 전체 채널 중 눈에 띄는 수치 2가지를 자동으로 강조 카드로 노출.

- 예시: "리뷰 점유율 1위 채널: 네이버 24.3%"
- 예시: "할인율 최고 채널: 옥션 24.6%"

분석 결과를 텍스트로 읽지 않아도 중요 포인트가 먼저 눈에 들어오도록 설계.

### 사용자 시나리오

> "채널마다 우리 상품 할인율이 얼마나 차이 나는지 비교해보고 싶다"  
> "리뷰 점유율이 가장 높은 채널이 어딘지 보고, 그 채널 상세로 바로 들어가고 싶다"

---

## 크로스채널 브랜드 검색

브랜드명을 입력하면 해당 브랜드 상품이 어느 채널에서 얼마에, 리뷰 몇 개와 함께 판매되는지 채널별로 묶어 보여준다.

**검색 조건:**
- 검색어를 입력한 뒤 "비교" 버튼 클릭 또는 Enter 키로 실행 (실시간 자동 검색 아님)
- 공백만 입력된 경우 검색 실행 안 됨
- 검색 대상: 상품명 기준 부분 일치 (대소문자 구분 없음)
- 결과는 채널별 그룹으로 표시 (채널명 → 해당 채널 내 상품 목록)
- 결과 상품 클릭 시 상품 상세 팝업 열림
- 결과 없을 경우 "해당 브랜드를 찾을 수 없습니다" 안내

**결과 정렬:** 채널 고정 순서 (네이버 → 쿠팡 → 올리브영 → 지마켓 → 11번가 → 카카오 → 옥션). 각 채널 내 상품은 검색어 일치 순 나열.

### 사용자 시나리오

> "설화수가 쿠팡이랑 네이버에서 각각 얼마에 팔리는지 한 번에 보고 싶다"  
> 브랜드명 입력 → 채널별 판매가·리뷰 비교 → 특이 채널 클릭해 상품 상세 확인

---

## 개별 채널 분석 (채널 선택 시)

### KPI 카드 4개

선택 채널에서의 아모레 성과를 핵심 수치 4개로 요약. 전월 대비 증감도 함께 표시.

| 지표 | 설명 |
|------|------|
| 아모레 상품 수 | 해당 채널 AP 등록 상품 수 |
| 리뷰 점유율 | 해당 채널 전체 화장품 중 AP 비율 |
| 평균 판매가 | AP 상품 평균 할인가 |
| 평균 할인율 | 정가 대비 평균 할인율 |

### 아모레 브랜드 순위표

해당 채널에서 AP 브랜드별 성과를 순위로 나열. 표 우측 상단에 카테고리 필터와 정렬 드롭다운이 있다.

| 항목 | 설명 |
|------|------|
| 브랜드 | 브랜드명 |
| 상품 수 | 해당 채널 내 AP 브랜드 등록 상품 수 |
| 총 리뷰 | 누적 리뷰 수 |
| 리뷰 점유율 | AP 전체 리뷰 중 해당 브랜드 비율. 1~3위 강조 |
| 평균 판매가 | 해당 채널 내 할인가 평균 |

**카테고리 필터 옵션:** 전체 카테고리 / 스킨케어 / 메이크업  
**정렬 옵션:** 리뷰 점유율순 / 상품수순 / 평균가 높은순

두 드롭다운은 AND 조건으로 동시 적용되는 것이 목표이나, **현재 프로토타입에서는 UI만 존재하고 실제 필터링·정렬 미동작.**

### 사용자 시나리오

> "쿠팡에서 우리 브랜드 중 어디가 가장 강한지 순위 보고 싶다"  
> "올리브영 점유율이 지난달보다 올랐는지 떨어졌는지 확인하고 싶다"  
> "스킨케어 카테고리만 보고 싶다" → 카테고리 필터 선택 (구현 예정)

---

## 화면 흐름

```
채널별 탭 진입
  ├── [전체] 선택
  │     ├── 채널 비교 표 확인
  │     ├── "상세 →" 클릭 → 해당 채널로 전환
  │     └── 브랜드 검색 → 상품 클릭 → 상품 상세 팝업
  └── [개별 채널] 선택
        ├── KPI 카드 확인
        └── 브랜드 순위표 확인
```

---

## 미결 / 추후 개선

- 채널별 브랜드 순위 데이터 현재 모든 채널에서 동일 → 실서비스 시 채널별로 분리
- 인사이트 카드 수치 현재 고정 → 실 데이터 기반 자동 계산으로 대체
- Excel 파일명 날짜 현재 고정 → 다운로드 시점 날짜로 자동 생성
- 카테고리·정렬 드롭다운 UI 존재하나 현재 미동작 → 추후 구현
