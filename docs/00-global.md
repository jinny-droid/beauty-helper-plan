# 공통 — 서비스 구조 및 전역 기획

---

## 서비스 개요

아모레퍼시픽 내부 직원이 7개 이커머스 채널(네이버·쿠팡·올리브영·지마켓·11번가·카카오·옥션)의 화장품 데이터를 한 화면에서 모니터링하는 인텔리전스 대시보드.

- 수집 주기: 일 1회 (전일 기준)
- 대상 채널: 7개
- 대상 브랜드: 아모레퍼시픽 16개 브랜드 + 주요 경쟁사

---

## 전체 화면 구조

```
로그인
  └── 메인 대시보드
        ├── Overview        전체 현황 요약
        ├── 성과 분석       채널별 · 브랜드별 · 카테고리별
        │     └── 브랜드 드릴다운
        ├── 경쟁사          경쟁 브랜드 모니터링
        └── 트렌딩          급상승 상품 모니터링
```

---

## 헤더 (전 화면 공통)

화면 상단에 항상 고정 노출.

- **로고:** 클릭 시 Overview로 이동
- **탭 네비게이션:** Overview / 성과 분석 / 경쟁사 / 트렌딩
- **데이터 기준일:** 현재 데이터 수집 기준일 표시
- **로그아웃:** 클릭 시 로그인 화면으로 이동

---

## 로그인

- 사내 계정 ID/PW 입력 후 접속
- 잘못된 계정 입력 시 에러 문구 안내
- 현재 프로토타입 계정: ID `amore` / PW `amore2026!`

> **기획 메모:** 현재 프로토타입은 진입 즉시 대시보드로 이동 (로그인 화면 스킵). 로그아웃 버튼으로 로그인 화면 확인 가능.

---

## 상품 상세 팝업 (전 화면 공통)

어느 화면에서든 상품명 또는 채널 셀을 클릭하면 팝업으로 열림.

### 사용자가 볼 수 있는 것

- 상품명 · 브랜드 · 카테고리 · 현재 판매가 · 리뷰 수
- 기간 선택 (1개월 / 2개월 / 3개월 / 전체)
- 가격 추이 차트: 선택 기간 내 정가와 할인가 변화
- 리뷰 추이 차트: 선택 기간 내 누적 리뷰 수 변화
- 히스토리 테이블: 날짜별 정가 / 할인가 / 리뷰 수 / 찜 수

### 활용 시나리오

- 리뷰가 갑자기 급증한 상품의 가격 변동 시점 확인
- 프로모션 기간과 리뷰·찜 증가 시점 연결해서 파악
- 특정 채널에서의 최근 가격 흐름 점검

### 기획 의도

팝업 진입 시 기본 기간은 '전체'로 설정해 전체 추세 먼저 제공. 이후 사용자가 단기 구간으로 좁혀 프로모션 효과 등을 확인하는 흐름.

---

## Excel 다운로드 (공통)

- **채널별 성과 탭:** 현재 화면의 채널 성과 데이터를 Excel로 다운로드
- **경쟁사 탭:** 현재 화면의 경쟁사 현황 데이터를 Excel로 다운로드

보고서 작성, 공유 등 오프라인 활용을 위한 기능.

## 공통 UI 규칙

| 규칙 | 설명 |
|------|------|
| AP 브랜드 강조 | 아모레퍼시픽 브랜드가 포함된 행은 파란 배경으로 구분. AP 배지 함께 표시 |
