# 경쟁사 탭 — 화면 기획

---

## 화면 목적

아모레퍼시픽과 경쟁하는 주요 브랜드들의 이커머스 동향을 모니터링한다.  
"어느 경쟁사가 요즘 치고 올라오고 있나", "어디서 신규 채널에 입점했나", "우리 점유율과 비교했을 때 경쟁사가 어떤 위치인가"를 채널 기준으로 확인하는 화면.

---

## 화면 구성

```
┌─────────────────────────────────────────┐
│  [채널 선택]                  [Excel]    │
├─────────────────────────────────────────┤
│  요약 카드 × 3                           │
│  (주시 필요 / AP 점유율 합산 / 하락 경쟁사) │
├─────────────────────────────────────────┤
│  경쟁 브랜드 현황표                       │
│  (상단: 브랜드 검색)                      │
└─────────────────────────────────────────┘
```

---

## 채널 선택

드롭다운으로 채널을 선택하면 표 전체와 요약 카드가 해당 채널 기준으로 즉시 갱신된다.

**선택 옵션:** 전체 채널 / 네이버 / 쿠팡 / 올리브영 / 지마켓 / 11번가 / 카카오 / 옥션

- 전체 선택 시: 전 채널 리뷰를 합산해 계산
- 개별 채널 선택 시: 해당 채널 내 리뷰만 기준으로 재계산

---

## 요약 카드 3개

화면 상단에서 현재 경쟁 상황을 세 가지 수치로 요약.

| 카드 | 설명 | 색상 |
|------|------|------|
| 주시 필요 브랜드 수 | 동향 감지 결과 "주시 필요" 등급을 받은 경쟁사 수 | 빨강 강조 |
| AP 합산 점유율 | 선택 채널 기준 AP 브랜드 전체 리뷰 점유율 합산 | 네이비 |
| 점유율 하락 경쟁사 수 | 전월 대비 점유율이 하락한 경쟁사 수 | 초록 (호재 신호) |

---

## 경쟁 브랜드 현황표

현재 모니터링 중인 경쟁사 10개 브랜드를 나열. AP 브랜드도 포함해 비교 맥락 제공.

### 브랜드 검색

상단 검색창에 브랜드명 일부를 입력하면 실시간으로 목록이 필터링됨.

**검색 조건:**
- 글자를 입력하는 즉시 필터링 (버튼 클릭 불필요)
- 브랜드명 부분 일치, 대소문자 구분 없음
- 앞뒤 공백 자동 제거 후 비교
- 검색어 입력 시 우측에 ✕ 버튼 노출 → 클릭 시 검색어 초기화
- 결과 없으면 "OO 브랜드를 찾을 수 없어요. 현재 모니터링 중인 브랜드 목록에 없습니다" 안내

### 테이블 정렬 기준

**기본 정렬:** 테이블 원본 데이터 순서 고정 (채널 변경해도 순서 유지). 사용자가 컬럼 클릭으로 정렬 변경하는 기능 없음.

### 테이블 컬럼

| 항목 | 설명 |
|------|------|
| 브랜드 | 브랜드명. AP 브랜드는 배지 표시 |
| 리뷰 점유율 | 막대 + 수치. AP: 네이비, 경쟁사: 로즈 |
| 점유율 전월 대비 | 증가 시 빨강, 감소 시 파랑, 동일 시 회색 |
| 총 리뷰 | 선택 채널 기준 누적 리뷰 수 |
| 리뷰 증감 | 전월 대비 리뷰 수 변화 |
| 상품 수 | 해당 브랜드 등록 상품 수 |
| 평균 판매가 | 해당 브랜드 평균 판매가 |
| 주요 동향 | 자동 감지된 동향 배지 |

AP 브랜드 행은 파란 배경으로 경쟁사 행과 시각적으로 구분.

---

## 동향 자동 감지

경쟁사별 리뷰·상품 수·채널 입점 현황 변화를 자동 분석해 "주요 동향" 컬럼에 배지로 표시.

### 감지 항목 (우선순위 순)

| 동향 | 조건 | 배지 색상 |
|------|------|----------|
| 다중 채널 신규 입점 · 급성장 | 신규 입점 채널 3개 이상 | 빨강 (주시) |
| [채널명] 신규 입점 | 기존에 없던 채널에 새로 입점 1~2개 | 빨강 (주시) |
| [채널명] 입점 확대 | 특정 채널 상품 수 2배 이상 증가 | 빨강 (주시) |
| [채널명] 중심 리뷰 급증 | 리뷰 25% 이상 급증 | 빨강 (주시) |
| 점유율 급등 · 주시 필요 | 점유율 2%p 이상 상승 | 빨강 (주시) |
| 점유율 상승 · 주시 | 점유율 1%p 이상 상승 | 빨강 (주시) |
| 점유율 하락세 | 점유율 0.3%p 이상 하락 | 초록 (호재) |
| [채널명] 비중 높음 | 단일 채널 집중도 40% 초과 + 리뷰 소폭 증가 | 회색 (정보) |
| 동향 이상 없음 | 위 조건 미해당 | 회색 (정보) |

"주시 필요" 등급이 붙은 경쟁사는 Overview 탭 > 경쟁사 주요 동향 카드에도 자동 노출됨.

## Excel 다운로드

현재 화면의 경쟁사 현황 데이터를 엑셀 파일로 저장.  
컬럼 포함: 브랜드명 / AP 여부 / 리뷰 점유율 / 전월 대비 / 총 리뷰 / 리뷰 증감 / 상품 수 / 평균 판매가 / 주요 동향

## 화면 흐름

```
경쟁사 탭 진입
  ├── 채널 선택 → 표 + 카드 전체 갱신
  ├── 브랜드 검색 → 실시간 필터링
  ├── 주요 동향 배지로 주시 경쟁사 파악
  └── Excel 다운로드 → 오프라인 보고서 활용
```
