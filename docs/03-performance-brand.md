# 성과 분석 > 브랜드별 + 드릴다운 — 화면 기획

---

## 화면 목적

아모레퍼시픽 16개 브랜드의 이커머스 성과를 브랜드 단위로 비교하고, 특정 브랜드를 선택해 상품·채널·시장 위치까지 깊이 파고드는 분석 흐름을 제공한다.

"브랜드별 점유율이 어떻게 되나" → "이 브랜드 상품이 각 채널에서 어떻게 팔리고 있나" 까지 연속적으로 이어지는 드릴다운 구조.

---

## 화면 구성

```
┌─────────────────────────────────────────┐
│  서브탭: [채널별] [브랜드별] [카테고리별]  │
├─────────────────────────────────────────┤
│  ↓ 브랜드 목록 화면                      │
│  16개 브랜드 성과 목록표                  │
├─────────────────────────────────────────┤
│  ↓ 브랜드 선택 후 드릴다운 화면           │
│  ← 브랜드 분석  [브랜드명]  AP           │
│  드릴다운 서브탭: [상품현황][채널성과][시장위치] │
│  (각 서브탭 콘텐츠)                       │
└─────────────────────────────────────────┘
```

---

## 브랜드 목록 화면

### 16개 브랜드 성과표

AP 16개 브랜드를 한 표에 나열해 리뷰 점유율 순으로 비교.

| 항목 | 설명 |
|------|------|
| 상품 수 | 전 채널 합산 (채널 중복 제거) |
| 리뷰 점유율 | AP 전체 리뷰 중 해당 브랜드 비율. 1~3위 강조 |
| 총 리뷰 | 전 채널 누적 리뷰 수 |
| 평균 판매가 | 전 채널 할인가 평균 |
| 채널 분포 | 네이버·쿠팡·올리브영·기타 비율을 4색 가로 막대로 표시 |

**정렬 기준:** 리뷰 점유율 기준 내림차순 고정. 사용자가 정렬 변경 불가, 별도 필터 없음.

- 행 전체를 클릭하면 해당 브랜드 드릴다운 화면으로 진입

### 사용자 시나리오

> "우리 16개 브랜드 중 이커머스 점유율 순위가 어떻게 되는지 한 번에 보고 싶다"  
> "설화수 클릭해서 어느 채널에서 얼마나 팔리는지 세부적으로 보고 싶다"

---

## 드릴다운 화면

브랜드 목록에서 특정 브랜드를 클릭하면 진입. 상단에 "← 브랜드 분석" 버튼으로 목록 화면으로 복귀 가능.

3개 서브탭으로 해당 브랜드를 다각도 분석:

---

### 서브탭 1 — 상품 현황

**목적:** 이 브랜드의 개별 상품이 각 채널에서 어떻게 분포하고 있는지 파악

#### 요약 KPI

| 지표 | 설명 |
|------|------|
| 전 채널 상품 수 | 해당 브랜드 등록 상품 총합 |
| 평균 리뷰 수 | 상품당 평균 누적 리뷰 |
| 평균 할인가 | 전 상품 평균 판매가 |

#### 필터

상품이 많을 때 원하는 상품을 빠르게 찾기 위한 필터 3종. **세 필터는 모두 AND 조건**으로 동시 적용된다. 예: "스킨케어 카테고리" AND "쿠팡" 선택 시 → 스킨케어이면서 쿠팡에 취급 중인 상품만 표시.

| 필터 | 옵션 | 검색/적용 조건 |
|------|------|--------------|
| 상품명 검색 | 텍스트 직접 입력 | 글자 입력마다 즉시 필터링. 상품명 부분 일치, 대소문자 구분 없음. 필터 변경 시 목록 첫 페이지(1~20번)로 자동 복귀 |
| 카테고리 선택 | 전체 카테고리 / 스킨케어 / 메이크업 / 헤어케어 / 바디케어 / 선케어 / 향수 / 네일 / 맨즈케어 | 선택 즉시 적용. 카테고리 완전 일치. 필터 변경 시 1페이지 복귀 |
| 채널 선택 | 전체 채널 / 네이버 / 쿠팡 / 올리브영 / 지마켓 / 11번가 / 카카오 / 옥션 | 선택 즉시 적용. 해당 채널에 취급 중인 상품만 표시. 선택 채널 컬럼 헤더 로즈 색 강조. 필터 변경 시 1페이지 복귀 |
| 초기화 버튼 | — | 세 필터 모두 "전체"로 한 번에 리셋 + 1페이지 복귀 |

> **채널 필터와 매트릭스의 관계:** 채널 필터로 특정 채널을 선택해도 다른 채널 컬럼이 사라지지 않는다. 행 목록이 "해당 채널에 취급 중인 상품"으로 좁혀지고, 선택 채널 헤더가 강조된다.

**정렬 기준:** 별도 정렬 옵션 없음. 현재 해당 브랜드 상품 목록 내 고정 순서 유지.

#### 상품 × 채널 매트릭스

행 = 상품, 열 = 7개 채널 (네이버 → 쿠팡 → 올리브영 → 지마켓 → 11번가 → 카카오 → 옥션)

각 교차 셀에는 해당 채널의 판매가와 리뷰 수를 표시.

- 취급하지 않는 채널은 "–" 로 표시
- 가장 저렴한 채널 셀은 로즈 색 테두리로 강조 (최저가 채널 바로 파악 가능)
- 셀 클릭 → 상품 상세 팝업

목록이 길면 "더 보기" 버튼으로 20개씩 추가 로드.

#### 사용자 시나리오

> "설화수 상품들이 쿠팡에서 어떤 가격에 팔리고 있는지 보고 싶다"  
> "특정 상품이 어느 채널에서 가장 저렴한지 빠르게 찾고 싶다"  
> "헤어케어 카테고리 상품만 보고 싶다" → 카테고리 필터 적용

---

### 서브탭 2 — 채널 성과

**목적:** 이 브랜드가 어느 채널에 집중되어 있고, 채널별 가격·리뷰 차이가 어떤지 파악

#### 요약 KPI

| 지표 | 설명 |
|------|------|
| 취급 채널 수 | 이 브랜드 상품이 등록된 채널 수 |
| 채널 총 리뷰 | 전 채널 리뷰 합산 |
| 최저가 채널 | 평균 판매가가 가장 낮은 채널명 |

#### 시각화

| 요소 | 설명 |
|------|------|
| 리뷰 점유율 스트립 | 채널별 리뷰 비중을 비례 가로 막대로 표시. 비중 10% 이상 채널은 막대 내부에 채널명 표시 |
| 채널별 평균 가격 | 가격 높은 순 수평 막대 비교 |
| 채널별 리뷰 수 | 리뷰 많은 순 수평 막대 비교 |

#### 사용자 시나리오

> "설화수가 어느 채널에 리뷰가 집중되어 있는지, 채널마다 가격 차이는 얼마나 나는지 보고 싶다"

---

### 서브탭 3 — 시장 위치

**목적:** 이 브랜드가 속한 카테고리에서 경쟁사 대비 어느 위치에 있는지 파악

#### 카테고리 내 리뷰 점유율 비교

해당 브랜드의 주력 카테고리 (상품 수 기준 자동 선정) 에서  
상위 5개 브랜드의 리뷰 점유율을 비례 가로 막대로 표시.

- 선택된 브랜드는 ★ 표시 + 네이비 색상으로 구분
- AP 브랜드와 경쟁사는 색상으로 구별
- 비중이 적은 나머지 브랜드는 "기타"로 묶음

#### 사용자 시나리오

> "설화수가 스킨케어 카테고리에서 경쟁 브랜드 대비 몇 위 정도 되는지 보고 싶다"

#### 미구현 (추후 개발)

- 가격 vs 리뷰 볼륨 2D 포지셔닝맵 (X축 가격대 / Y축 리뷰 수)
- 가격대 분포 바 차트

## 화면 흐름

```
브랜드별 탭 진입
  └── 브랜드 목록 확인
        └── 브랜드 행 클릭 → 드릴다운 진입
              ├── [상품현황] 매트릭스 확인
              │     └── 셀 클릭 → 상품 상세 팝업
              ├── [채널성과] 채널 분포·가격·리뷰 비교
              └── [시장위치] 카테고리 내 경쟁 위치 확인
              └── "← 브랜드 분석" 클릭 → 목록 복귀
```

---

## 미결 / 추후 개선

- 드릴다운 상품 수가 현재 적음 (프로토타입 샘플 데이터) → 실 데이터 연동 시 해소
- 채널 분포 막대 현재 픽셀 직접 지정 → 비율 기반 자동 계산으로 변경 권장
- 시장 위치 탭 포지셔닝맵·가격대 분포 미구현 → 추후 Chart.js scatter 도입 검토
